# Metro Mayhem

Playable HTML5 Canvas platformer prototype with six city-themed levels, original canvas-drawn characters, ghost enemies, projectiles, moving platforms, HUD, score, lives, and level progression.

## Run locally

Open [index.html](/Users/fernando/Documents/New project/game/index.html) in a modern browser, or serve the folder with a simple static server:

```bash
cd /Users/fernando/Documents/New\ project/game
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Controls

- `Left Arrow` / `Right Arrow`: move
- `Space`: jump
- `X`: shoot
- `Shift`: run faster

## Mobile

- Landscape is the preferred mobile play mode
- Use the left thumb buttons to move
- Tap `Jump` for jump and double-jump
- Tap `Fire` to shoot
